# ROBO.DANCE

Hilarious meme website with a high-quality dancing robot.

## Features
- Live dance stage (groove / spin / break / vibe)
- Meme forge — caption, download PNG, post to wall
- Reaction buttons + chaos meter
- Local meme wall
